# lab3_bd

Lab 3: OLTP операції
Опис: Демонстрація операцій маніпулювання даними: SELECT, INSERT, UPDATE, DELETE.
Файл
lab3.sql

Структура скрипта:
1. SELECT запити (10 запитів)

Прості SELECT з різними умовами WHERE
SELECT з LIKE для пошуку
SELECT з JOIN (2-4 таблиці)
SELECT з підзапитами
SELECT з обчисленнями (CASE, дати)

2. INSERT операції (7 операцій)

INSERT з явними значеннями
INSERT з підзапитами (щоб уникнути жорстко закодованих ID)
INSERT множинних записів

3. UPDATE операції (5 операцій)

UPDATE одного запису
UPDATE з WHERE умовами
UPDATE з обчисленнями
UPDATE з CASE
UPDATE множинних записів

4. DELETE операції (5 операцій)

DELETE з WHERE умовами
DELETE з підзапитами
Безпечне видалення з перевіркою зовнішніх ключів

5. Транзакції
Всі модифікуючі операції обгорнуті в транзакції (BEGIN/COMMIT)
